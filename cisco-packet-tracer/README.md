# Simulasi Jaringan Cisco Packet Tracer

Dokumentasi project simulasi jaringan kantor menggunakan Cisco Packet Tracer.

## Tujuan

- Mendesain topologi jaringan sederhana yang mudah dipahami.
- Mengatur IP address dan konektivitas antarperangkat.
- Melakukan pengujian koneksi menggunakan ping.
- Melatih pemahaman dasar routing, switching, dan DHCP.

## Perangkat simulasi

- Router
- Switch
- PC client
- Server
- Kabel straight-through/crossover sesuai kebutuhan simulasi

## Ruang lingkup konfigurasi

- Perancangan topologi star network.
- Pembagian alamat IPv4.
- Konfigurasi DHCP dasar.
- Pengujian konektivitas antar-client dan server.

## Hasil yang diharapkan

- Semua client mendapatkan konfigurasi jaringan yang benar.
- Client dapat saling terhubung sesuai skenario.
- Dokumentasi konfigurasi dapat dikembangkan menjadi file `.pkt` ketika file simulasi sudah siap diupload.

## Catatan pengembangan

Jika file Packet Tracer sudah tersedia, simpan file `.pkt` di folder ini agar tombol GitHub pada portfolio langsung mengarah ke dokumentasi dan file project.
