Simulasi Jaringan Cisco Packet Tracer

Project simulasi jaringan untuk memahami topologi, pembagian IP, DHCP, switching, routing dasar, dan pengujian konektivitas antar perangkat.

Buka GitHub Lihat Project Lain

Ringkasan

Simulasi ini dibuat untuk melatih alur perancangan jaringan dari topologi sampai validasi koneksi. Setiap perangkat diposisikan sesuai kebutuhan dasar jaringan kantor kecil.

  • Mendesain topologi jaringan terstruktur.
  • Melakukan konfigurasi IPv4 dan DHCP.
  • Menguji koneksi dengan ping dan connectivity test.
01

Topologi

Menggunakan pendekatan star network agar perangkat mudah dipetakan dan diuji.

02

Konfigurasi

IP addressing, DHCP, router, switch, dan client dikonfigurasi agar saling terhubung.

03

Pengujian

Validasi dilakukan melalui ping dan pengecekan konektivitas antar perangkat.

04

Dokumentasi

Catatan project disimpan agar konfigurasi bisa dipelajari kembali lewat GitHub.
