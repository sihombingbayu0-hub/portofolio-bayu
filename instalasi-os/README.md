# Instalasi & Konfigurasi Sistem Operasi

Dokumentasi project IT Support untuk instalasi sistem operasi Windows dari media bootable sampai perangkat siap digunakan.

## Tujuan

- Menyiapkan media instalasi menggunakan bootable USB.
- Melakukan instalasi sistem operasi Windows.
- Mengatur driver, pembaruan sistem, dan aplikasi pendukung.
- Memastikan perangkat siap digunakan untuk aktivitas harian.

## Tahapan kerja

1. Backup data penting sebelum instalasi.
2. Membuat bootable USB.
3. Mengatur boot priority melalui BIOS/UEFI.
4. Melakukan instalasi Windows.
5. Menginstal driver utama.
6. Melakukan update sistem.
7. Menginstal aplikasi pendukung.
8. Mengecek audio, jaringan, display, storage, dan performa dasar.

## Checklist hasil akhir

- Sistem operasi berhasil boot normal.
- Driver perangkat terbaca dengan baik.
- Jaringan dan audio aktif.
- Aplikasi dasar terinstal.
- Perangkat siap digunakan.

## Catatan pengembangan

Tambahkan foto proses instalasi, screenshot Device Manager, atau catatan troubleshooting jika dokumentasi lapangan sudah tersedia.
